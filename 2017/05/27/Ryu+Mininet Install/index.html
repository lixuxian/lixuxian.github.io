Ryu+Mininet安装使用向导

Ryu的安装及使用

安装：

在终端中输入如下命令：
$ wget https://raw.githubusercontent.com/sdnds-tw/ryuInstallHelper/master/ryuInstallHelper.sh
$ bash ryuInstallHelper.sh

使用：

`$ ryu-manager yourapp.py`

若无yourapp.py则默认启动一个OpenFlow Controller
成功启动界面：

Mininet的安装及使用

安装：

从git下载源码
git clone git://github.com/mininet/mininet

安装mininet：
./mininet/util/install.sh -s mydir -nfv
[options]:
-a: 全部安装
-nfv: 仅安装MINIENT OPENFLOW引用多SWITCH 和OPEN VSWITCH
-s mydir: 安装到指定目录
使用：

启动mininet：
$sudo mn
默认启动一个switch两个host
界面如下：

使用Ryu作为控制器时，启动mininet的命令为：
$ sudo mn --ontroller remote
mn的—topo参数可以快速建立拓扑结构或者使用义拓扑，如：
$ sudo mn --controller remote --topo linear,3
建立了带有三个switch的线性拓扑

Mininet基本命令：

nodes            查看全部节点
links            查看链路信息
dump            输出各节点的信息
h1 ping h2    测试主机之间的连通性
iperf            两个节点之间用指定简单的 TCP 测试
iperfudp        两个节点之间用指定简单的 udp测试
noecho            运行交互窗口，关闭回应
pingpair            两个主机将互 ping
help            列出命令文档，查看命令帮助： help command 
exit/quit            退出 mininet 命令行
h1 ifconfig     查看当前那主机的 ip 地址，如： h1 ifconfig
dpctl             对交换机的配置管理的相关操作，如：# dpctl dump-flows tcp:127.0.0.1:6634
py/sh            执行 python 表达式或运行外部 shell 程序
